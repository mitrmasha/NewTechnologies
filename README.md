# Система управления заявками партнеров

## Описание проекта
Разрабатываемая система предназначена для автоматизации процессов обработки заявок партнеров компании, занимающейся производством и реализацией продукции через партнерскую сеть. Система обеспечивает эффективное управление данными о продукции и заявках в цифровом формате.
#### Функциональность формы заявки:
- Тип партнера (выпадающий список)
- Наименование компании
- ФИО директора
- Адрес
- Рейтинг (целое неотрицательное число)
- Телефон
- Email
